🎮 Projects

Desolate Place

Desolate Place is my diploma project, which I have worked on in a team of four people, a low-poly 3D game in Unity. We used Agile methodology, planning our tasks in one-week sprints in Jira and presenting our weekly progress during classes. The game combines sci-fi puzzle gameplay with elements of atmospheric horror.

  • Game progression design
  • Player experience design (game flow)
  • Level design
  • Puzzles design and implementation
  • User interface (UI) and heads-up display (HUD) design and implementation
  • Creating shaders for UI and HUD elements
  • 3D modeling of some elements

Law of Attraction

Law of Attraction is a 2D platformer game created as part of a GameJam event at PJATK University. The theme was "Up Up and Away." Our team of 4 worked under a 20-hour time constraint and took 3rd place. In the game, players must go up using a gun that shoots two types of projectiles, which carry different gravitational forces: attraction and repulsion. Players navigate by dodging turrets that fire at them while using these gravitational forces to climb in ways that would otherwise be impossible.

  • Gameplay design
  • Developing the gravitational projectiles mechanic and their interactions with the player and objects
  • Implementing movable platforms
  • Designing and implementing the user interface (UI)

3D Tower Defense in Unity

This is the second project which I've worked on with the same team of four as in my diploma project. This game is a simple tower defense game, and the focus of the project is to learn and apply common design and architectural patterns, which is why it is not polished visually.

  • Implementing the grid and placement system
  • Level design
  • User interface (UI) and heads-up display (HUD)
  • Creating the tutorial

Treasure Hunters

This was my first project in Unity, with the main goal of learning the engine and gaining experience working on a team project. The team consisted of 4 people, and we worked on the project for 6 weeks. We used free assets from PixelFrog's Treasure Hunters pack and implemented all game logic according to a Game Design Document (GDD) template.

  • Implementing obstacles like moving/collapsing platforms and spikes
  • Level design
  • User interface (UI) and heads-up display (HUD) design and implementation
  • Respawn and checkpoints implementation
